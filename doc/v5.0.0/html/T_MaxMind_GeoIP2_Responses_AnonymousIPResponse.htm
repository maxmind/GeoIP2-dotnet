Properties
  NameDescription
IPAddress
The IP address that the data in the model is for. If you performed a "me" lookup against the web service, this will be the externally routable IP address for the system the code is running on. If the system is behind a NAT, this may differ from the IP address locally assigned to it.
IsAnonymous
Returns true if the IP address belongs to any sort of anonymous network.
IsAnonymousVpn
Returns true if the IP address is registered to an anonymous VPN provider.
IsHostingProvider
Returns true if the IP address belongs to a hosting or VPN provider (see description of IsAnonymousVpn property).
IsPublicProxy
Returns true if the IP address belongs to a public proxy.
IsResidentialProxy
This is true if the IP address is on a suspected anonymizing network and belongs to a residential ISP.
IsTorExitNode
Returns true if IP is a Tor exit node.
Network
The network associated with the record. In particular, this is the largest network where all of the fields besides IPAddress have the same value.
